# Sales Data Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis on sales data to derive business insights such as revenue distribution, product performance, and regional sales analysis.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## 📊 Analysis Performed
- Data inspection and cleaning
- Revenue calculation
- Region-wise revenue analysis
- Product-wise sales analysis
- Statistical summary
- Visualization (Bar chart)

---

## 📈 Output
The project generates:
- Total Revenue
- Revenue by Region
- Units Sold by Product
- Revenue Visualization (region_revenue.png)

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python sales_eda.py
